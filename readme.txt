说说使用linux记事本的部分技巧：
  字体颜色：linux的“文本编辑器”里，点“菜单栏”-“查看”-“突出显示模式”-“源代码”-“java”
  这样就会把记事本的java代码部分突出显示出来，好看很多
  笔记中，也有"html"、“javascript”等的显示模式
  “菜单栏”-“编辑”-“首选项”，可设置显示效果

换行：
  linux记事本的换行，在windows看的时候，会变成一个小黑方块，很难看
  可以在linux的“文本编辑器”，选“菜单“-”搜索“-“替换”  也可用快捷键：Ctrl+H
  搜索填：“\n”  替换为填：“\r\n”  再点“全部替换”，这样的txt，在linux和windows都可以正常显示换行

生成PDF：
  linux的“文本编辑器”，点“打印”
  “打印机”选“Create a PDF document”，再点“打印”。就把txt生成pdf
  还可以点“菜单栏”-“文件”-“页面设置”，设置打印效果
